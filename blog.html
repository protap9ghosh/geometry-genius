What is DOM and what is the purpose of DOM?

DOM (Document Object Model) is a set of functions used to modify webpages. DOM represents our written code (HTML or XML) in such a way that it can be easily modified with programming languages. Represents the document itself in an object-oriented way so that we can interact with the page using DOM.

The purpose of the DOM is to: creating elements, Remove an old element, To modify an old element, Modify, add, remove attributes, add styling, Create events on any element.

What is event bubble?

Event bubbling is much like water bubbling. Event bubbles are a method of propagating events in the HTML DOM API. It is a process that starts with the element that triggers the event and then bubbles up to the elements in the sequence.

What is a callback function and why will you use it?

The job of function is to create a function and use that function repeatedly without writing the same code again and again. In general, a function can return another function and at the same time a function can take another function as an argument. And such a function takes another function as an argument, then we call the function passing to that argument as a callback function.
